# JWorld-Metaverse

- 메타버스 SNS 클라이언트사이드 만들기
- 싸이월드 모티브

### 클라이언트사이드 기술스택

- [x] Next.js
- [x] TypeScript
- [x] Three.js (React-Three-Fiber)
- [x] Tailwind CSS
- [x] Styled-Components
- [ ] GraphQL
- [ ] GraphQL-Apollo
- [ ] REST API (Axios)
- [x] Socket.io
- [ ] WebRTC
